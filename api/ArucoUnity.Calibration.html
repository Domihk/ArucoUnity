    Namespace ArucoUnity.Calibration

    ArucoCameraCalibration

    Calibrates a ArucoCamera with a ArucoBoard and saves the calibrated camera parameters in a file managed by ArucoCameraParametersController. Base class to reference in editor fields.

    See the OpenCV and the ArUco module documentations for more information about the calibration process: http://docs.opencv.org/3.3.0/da/d13/tutorial_aruco_calibration.html and https://docs.opencv.org/3.3.0/da/d13/tutorial_aruco_calibration.html
