Detailed Description

Calibrate a ArucoObjectDetector.ArucoCamera camera system with a ArucoBoard, and save the calibration results in a file to be used for tracking.

See the OpenCV documentation for more information about the calibration: http://docs.opencv.org/3.2.0/da/d13/tutorial_aruco_calibration.html

§ Calibrate()

void ArucoUnity.Controllers.ArucoCalibrator.Calibrate ( )
inline

Calibrate each camera of the ArucoObjectDetector.ArucoCamera system using the detected markers added with AddCurrentFrameForCalibration(), the CameraParameters, the CalibrationFlagsController and save the results on a calibration file. Stereo calibrations will be additionally executed on these results for every camera pair in StereoCalibrationCameraPairs.
