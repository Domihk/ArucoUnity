Perform omnidirectional camera calibration, the default depth of outputs is CV_64F.

Parameters
objectPointsVector of vector of Vec3f object points in world (pattern) coordinate.
imagePointsVector of vector of Vec2f corresponding image points of objectPoints.
imageSizeImage size of calibration images.
cameraMatrixOutput calibrated camera matrix.
xiOutput parameter xi for CMei's model.
distCoeffsOutput distortion parameters.
rvecsOutput rotations for each calibration images.
tvecsOutput translation for each calibration images.
flagsThe flags that control calibrate.
criteriaTermination criteria for optimization.
idxIndices of images that pass initialization, which are really used in calibration.
exceptionThe first exception threw by any trigerred CV_ASSERT.
Returns
The overall RMS re-projection error.
