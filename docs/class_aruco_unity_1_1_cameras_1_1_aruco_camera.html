Detailed Description

Capture and display every frame the images of any camera system with a fixed number of cameras to use for calibration or ArUco object tracking. Each camera of the system is associated with a Unity camera that shots as background the current captured frame.

If you want to use a custom physical camera not supported by Unity, you need to derive this class. See ArucoCameraWebcam as example. You will need to implement StartCameras, StopCameras and Configure and to set ImageDatas when UpdateCameraImages is called.
