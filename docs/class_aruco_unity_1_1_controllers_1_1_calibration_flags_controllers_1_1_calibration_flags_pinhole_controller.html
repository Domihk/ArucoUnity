Detailed Description

Manages flags for the calibration process of pinhole cameras.

See the OpenCV documentation for more information about these calibration flags: http://docs.opencv.org/3.2.0/d9/d0c/group__calib3d.html#ga3207604e4b1a1758aa66acb6ed5aa65d
