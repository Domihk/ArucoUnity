Detailed Description

Detect markers, display results and place game objects on the detected markers transform.

Member Function Documentation

◆ ApplyTracking()

IEnumerator ArucoUnity.ArucoTracker.ApplyTracking ( )
inlineprotected

Draw the results of the detection and place each detected ArUco object on the ArucoObjectsController.ArucoObjects list, according to the results of the tracking thread.

In a coroutine to be executed after the ArucoCamera.Images update on the Update() function, but before the ArucoCamera.ImageTextures update on the LateUpdate() function. See: https://docs.unity3d.com/Manual/ExecutionOrder.html.
